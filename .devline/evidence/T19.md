- `web/app/portal-data.ts`：繳租確認管理 → href `/billing`、status `available`；例外費用管理維持 `development`（尚無頁）。
- `web/app/providers.tsx`：topnav 已含「電費作業」→ `/billing`（可達）。

## 指揮方修正（一行型別收斂 bug）
- `page.tsx` `formatScope` 尾行用 `scope.room_ids`，但 TS 控制流 narrow 後殘留 `{type:"all"}`，該型別無 `room_ids` → `tsc` 報 TS2339，`next build` 會擋。中斷 Codex 未跑到 build 就被殺，漏掉。
- 修正：補明確 `"room_ids" in scope` narrow 分支 + 安全 fallback `return "全部"`。
- 此修正並非只為過 build：live 驗證顯示後端把 `{room_id:4}` **正規化存成 `{"room_ids":[4]}`**，GET run 回傳即走 `room_ids` 分支，顯示邏輯本就需要它。
